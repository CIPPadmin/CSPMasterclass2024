# Power Automate Learning Resources

Here are some useful links to get started with and learn more about Power Automate:

## Official Microsoft Resources

*   **Microsoft Learn - Power Automate:** The main hub for documentation, learning paths, and modules.
    https://learn.microsoft.com/en-us/power-automate/

*   **Power Automate Documentation:** Specific documentation covering various aspects of Power Automate.
    https://learn.microsoft.com/en-us/power-automate/getting-started

*   **Power Automate Blog:** Official news, updates, and articles from the Power Automate team.
    https://powerautomate.microsoft.com/en-us/blog/

*   **Microsoft Power Automate YouTube Channel:** Official videos, tutorials, and feature showcases.
    https://www.youtube.com/@MicrosoftPowerAutomate
